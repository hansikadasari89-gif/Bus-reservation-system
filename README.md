# 🚌 Bus Reservation System

A Python-based Bus Reservation System that enables users to book, manage, and view bus tickets through an intuitive graphical user interface. The application uses a database to securely store reservation details and provides an efficient ticket booking experience.

## 🚀 Features

- 🚌 Book bus tickets
- 👤 Passenger registration and details
- 🎫 Seat reservation system
- ❌ Cancel bookings
- 📋 View reservation records
- 💾 Database integration for secure data storage
- 🖥️ User-friendly graphical interface

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- SQLite
- Object-Oriented Programming (OOP)



```bash

```

## 💡 How It Works

1. Launch the application.
2. Enter passenger details.
3. Select the desired bus and seat.
4. Confirm the booking.
5. View or cancel reservations anytime.
